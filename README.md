## How to Run

### Machine Learning Projects

```bash
# Clone the repository
git clone https://github.com/EKtechie/data-projects.git
cd data-projects

# Install dependencies
pip install -r requirements.txt

# Launch a notebook
jupyter notebook machine-learning/fraud-detection/fraud_detection.ipynb
```

### Tableau Projects

1. Download the `.twbx` file from the relevant project folder
2. Open with **[Tableau Public](https://public.tableau.com)** (free) or **Tableau Desktop**
3. Interact with filters and views — datasets are embedded in the workbook
